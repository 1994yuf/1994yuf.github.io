# 02 vuepress 自动生成 nav 和 sidebar

VuePress 边栏和导航栏生成器基于文件和目录结构。集中您的文档，而不是侧边栏或导航栏。

# 概述

# 安装

cnpm install -D boboidream/vuepress-bar

yarn install boboidream/vuepress-bar

# 用法

添加vuepress-plugin-autobar在您的网站或主题的配置文件。查看有关使用插件的官方文档 (opens new window)。

// .vuepress/config.js
// or
// .vuepress/theme/index.js
module.exports = {
  plugins: ['autobar']
}

# 特点

  • 创建导航栏和侧边栏：navbar在目录中添加前缀，例如nav.guide 或 nav.01.guide
  • 自定义排序：带数字前缀的目录，或将ordermeta添加到文件中，例如01.guide
  • 多个侧边栏
  • 没有配置
  • 将 README.md 添加到第一个可用组，例如VuePress网站。（可以通过选项禁用）
  • 可以在目录名称中传递参数。（请参见下面的高级示例。）

# 例子

# 使用导航栏

|- docs/
  |- .vuepress
  |- nav.01.guide/
    |- README.md
  |- nav.02.api/
    |- classes/
      |- member.md

{
  nav: [
    { text: 'Guide', link: '/nav.01.guide/' },
    { text: 'Api', link: '/nav.02.api/' }
  ],
  sidebar: {
    '/nav.01.guide/': [ '' ],
    '/nav.02.api/': [ { title: 'Classes', children: [ '', 'classes/member' ] } ]
  }
}

# 没有导航栏

|- docs/
  |- .vuepress
  |- 01.guide/
    |- README.md
  |- 02.api/
    |- classes/
      |- member.md

{
  nav: [],
  sidebar: [
    { title: "Guide", children: ["01.guide/"] },
    {
      title: "Api",
      children: [{ title: "Classes", children: ["02.api/classes/member"] }]
    }
  ]
};

# 进阶

# 约定

可以在目录名称中传递侧边栏参数。您可以在双破折号 -- 之间用逗号分隔后再传递以下参数：

  • nc 设置 collapsable 为 false。
  • dX 设置 sidebarDepth 为 X。
|- docs/
  |- 01.guide--nc,d2/
    |- README.md

{
  nav: [],
  sidebar: [
    {
      title: "Guide",
      collapsable: false,
      sidebarDepth: 1,
      children: ["01.guide--nc,d2/"]
    },
  ]
};

VuePress 需要链接中的 README.md 默认文件 navbar。忘记 README.md 将跳过该导航栏项目的创建。

# 高级选项

// .vuepress/config.js
// or
// .vuepress/theme/index.js
const autobar_options = {
  rootDir: 'xxx',
  stripNumbers = true,
  maxLevel = 2,
  navPrefix = "nav",
  skipEmptySidebar = true,
  skipEmptyNavbar = true,
  multipleSideBar = true,
  setHomepage = 'hide' | 'toGroup' | 'top'
};

module.exports = {
  plugins: [
    [ 'autobar', autobar_options ]
  ]
}

如果您想使用简单的链接，也许您会喜欢 vuepress-plugin-rpurl (opens new window)。

# API

参数 类型 默认 描述
rootDir String ${ctx.sourceDir} 文档所在的根目录。
stripNumbers Boolean true 从目录名称中删除数字前缀，这有助于进行排序。
maxLevel Number 2 子目录遍历的最大递归级别。
navPrefix String nav 导航栏和多边工具条目录的前缀。
skipEmptySidebar Boolean true 如果目录为空，请不要将项目添加到侧栏。
skipEmptyNavbar Boolean true 如果目录为空，请勿将项目添加到导航栏。
MultipleSideBar Boolean true 如果有导航栏项，则创建多个侧栏。
setHomepage String hide hide：在侧边栏中隐藏主页链接。
toGroup：将README.md添加到侧边栏的第一组中。
top：在侧边栏顶部显示主页链接
拼音导航 Boolean false 不要将中文导航翻译成拼音。

pinyinNav：true，应与 vuepress-plugin-permalink-pinyin (opens new window) 一起使用。
